Objectives:

 

To understand modern web technologies and build Web Mashup using existing open API’s (Application Programming Interfaces).

 

Project description:

This project is an AJAX driven Mashup which utilizes API’s(Application Programming Interfaces) of various sites to integrate data on the web. The Mashup contains several widgets which load content from data sources such as Twitter Search, Yahoo weather API and RSS feeds of various blogs. The technologies used in this project are HTML5, JavaScript, AJAX and YQL (Yahoo Query Language). The project start page is designed using simple HTML and CSS. Widgets can be dynamically added and removed using JavaScript and they can be populated with data asynchronously using AJAX. To invoke various APIs (Application Programming Interfaces) easily, YQL (Yahoo Query Language) was used. YQL helps in querying any URL in a syntax similar to SQL (e.g., select * from twitter.search) and output data in JSON, a light weight data interchange format. Some of the new features of HTML5 such as Local storage, inline editing have been used to enhance the richness of the application.

This project helped me in understanding the intricacies of the modern web such as APIs, asynchronous calls, some of the features in HTML5 and working with public APIs.
